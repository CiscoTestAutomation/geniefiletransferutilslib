GenieFileTransferUtilsLibs
==========================

The Genie FileTransferUtils package (or simply ``filetransferutils``) contains
a collecton of file transfer libraries that can are used for file transfer
to/from device/remote servers.

The Genie FileTransferUtils package is an extention of the top-level Cisco Genie
automation package, based on Cisco pyATS framework.

This package was initially developed internally in Cisco, and is now
available to the general public starting late 2017 through :devnet:`Cisco DevNet <http>`.

For more details, refer to :pyats:`pyATS <http>`.
